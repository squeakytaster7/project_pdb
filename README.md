Aplikasi web-based dashboard ini menampilkan peta heatmap PDB (GDP) seluruh dunia berdasarkan data World Bank API.
Kamu bisa memilih antara:

GDP total (NY.GDP.MKTP.CD)

GDP per capita (NY.GDP.PCAP.CD)

Serta mengatur tampilan log scale, filter region, dan filter tingkat pendapatan.
Hasil peta bisa diunduh dalam format CSV dan HTML interaktif.
